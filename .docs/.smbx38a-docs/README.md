
# 概述

本目录主要存放 SMBX 38A 引擎的文档

## 目录结构

- `./.docs/.smbx38a-docs/NPC List.md` - 存放 NPC 列表, 用于查询 ID 对应的具体 NPC
- `./.docs/.smbx38a-docs/smbx-38A File specifications (LVL WLD WLS).md` - 存放 SMBX 38A 文件格式规范

## 引擎相关

假设 `${Editor}` 是引擎 exe 所在的路径, 那么:
- `${Editor}/smbx.exe` 是引擎 exe
- `${Editor}/graphics/` 是引擎 graphics 目录, 里面包含了所有内建的 sprite, 你可以通过目录工具来查找这些 sprite
- `${Editor}/music/` 是引擎 music 目录, 里面包含了所有内建的 music, 你可以通过目录工具来查找这些 music
